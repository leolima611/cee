## para actualizar local
usar los siguientes comandoa antes de iniciar tu trabajo
- git fetch
- git merge

## para subir tus cambios
para la actualizacion de el repo no olvidar los comandos: 
- git status  
- git add .  
- git commit -m "ingresa un numero de version"
- git push
